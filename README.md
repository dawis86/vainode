## 💻 Tehniskais apraksts

Vietne ir veidota bez ietvariem (*framework-less*), izmantojot tīras un modernas tīmekļa tehnoloģijas, lai nodrošinātu maksimālu ātrdarbību un vieglu uzturēšanu.

*   **Pamatā:** `HTML5`, `CSS3`, `JavaScript (ES6 Modules)`.
*   **Struktūra:**
    *   `index.html`: Galvenais aplikācijas "apvalks".
    *   `css/`: Modulāra CSS arhitektūra (`variables.css`, `global.css`, `components.css`, `sections.css`).
    *   `js/`: Modulāra JavaScript arhitektūra (`app.js`, `router.js`, `feed.js` u.c.).
    *   `jpg/`: Vizuālie materiāli.
*   **Dizaina principi:** Mobile-first, adaptīvs dizains ar `Grid` un `Flexbox`, plūstošas animācijas (`cubic-bezier`), augsts kontrasts un lasāmība.

## 🚀 Projekta palaišana

Šis ir statisks projekts. Lai to apskatītu lokāli, vienkārši atveriet `index.html` failu jebkurā modernā pārlūkprogrammā. Lai nodrošinātu pareizu ES moduļu darbību, ieteicams izmantot vietējo serveri, piemēram, `Live Server` paplašinājumu Visual Studio Code.
